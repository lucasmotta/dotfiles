# dotfiles

```bash
git clone https://github.com/lucasmotta/dotfiles.git
cd dotfiles
./install.sh
```

## Manual config

### Atom
- Install [sync-settings](https://github.com/atom-community/sync-settings)
- Configure it using secret gist config
- Run `Sync Settings: Restore`
